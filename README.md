# Signal Restore

## Local Expert Support for Business Phone Systems

Signal Restore provides independent local support for business phone systems, VoIP, UCaaS, CCaaS, call routing, provider coordination, cloud migrations, troubleshooting, and communications management for businesses across Southern California.

We help businesses reduce downtime, improve customer communication, and simplify the complexity of modern phone systems.

From small offices to multi-location operations, we work directly with business owners, office managers, IT teams, and providers to make communications reliable, scalable, and easier to manage.

# Assets Structure

## Required Assets Folder

```text
/assets
  signal-restore-logo.png
  live-agent.png
  projects-infographic.png
  industries-infographic.png
  modern-communications-remote.png
```

### Important

Image filenames must match exactly.

GitHub Pages is case-sensitive and space-sensitive.

Example:

Correct:

```text
projects-infographic.png
```

Incorrect:

```text
   projects-infographic.png
```

Leading spaces or incorrect capitalization will break images.

# Deployment Notes

## GitHub Pages

This project is designed for deployment through GitHub Pages.

### Important Rules

Use relative asset paths:

Correct:

```html
<img src="assets/projects-infographic.png">
```

Avoid:

```html
<img src="./assets/projects-infographic.png">
```

or incorrect paths with hidden spaces.
